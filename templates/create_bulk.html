Collez une liste JSON d'objets recto_text / verso_text. Les cartes sont ajoutées en boîte 1 et étalées dans le temps (max {{ max_per_day }}/jour, à partir de demain) pour ne pas surcharger une seule journée. Tout‑ou‑rien : si une entrée est invalide, rien n'est importé et l'erreur est signalée.

Voir un exemple de format 
[
  {
    "recto_text": "Lucratif",
    "verso_text": "Qui rapporte de l'argent"
  },
  {
    "recto_text": "Castor",
    "verso_text": "Beaver"
  }
]

Optionnel pour une face image : recto_path / verso_path. Accepte une URL d'image (http(s)://…) ou un chemin vers un fichier de votre dossier images (ex. images/chat.png ou simplement chat.png). Si présent, il remplace le texte de cette face. Déposez les fichiers correspondants dans la zone « Images » ci‑dessous : ils sont enregistrés dans le dossier images sous leur nom d'origine, et chaque chemin local doit pointer vers un fichier réellement présent.
